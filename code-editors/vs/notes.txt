--crear snippet
1. seleccionar el codigo que quieres agregar como snippet y click-derecho/export-as-snippet
2. Aparecera todo tu codigo en una pantalla nueva, agregar el shortcut como mandaras llamar el snippet
3. seleccionar el nombre del shortcut que pusiste y darle copy
4. darle a guardar el snippet
5. en la pantalla de donde quieras guardar el snippet borra el nombre default y ponle el mismo nombre que usaste para el shortcut
6. listo, ya puedes usar tu snippet
